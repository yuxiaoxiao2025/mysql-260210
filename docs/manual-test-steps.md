## 🐛 常见问题

### Q1: 启动时报错 "DASHSCOPE_API_KEY not found"

**解决方案:**
```bash
# 检查 .env 文件
cat .env | grep DASHSCOPE

# 如果没有，添加:
echo "DASHSCOPE_API_KEY=sk-xxxxxx" >> .env
```

### Q2: 查询返回 "未匹配到操作模板"

**说明:** 这是正常的，系统会 fallback 到 LLM 生成 SQL。只要 LLM 生成的 SQL 正确即可。

### Q3: 响应时间过长 (> 5秒)

**可能原因:**
- 网络延迟
- API 限流

**解决方案:**
- 检查网络连接
- 等待一段时间后重试
