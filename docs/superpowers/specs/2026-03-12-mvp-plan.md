## 三、MVP 架构

```
用户输入
    │
    ▼
┌─────────────────────────────────────────┐
│         ReACT Orchestrator (简化版)      │
│                                         │
│  1. 模型推理 → 调用工具                  │
│  2. 执行 → 返回结果                      │
│  3. 用户修正 → 重新执行                  │
│                                         │
│  （无记忆学习）                          │
└─────────────────────────────────────────┘
    │
    ▼
工具层（4个工具）
├── search_schema(query)
├── execute_sql(sql)
├── list_operations()
└── execute_operation(op_id, params)
```
